## Question

**What is the difference between a git merge and a GitHub pull request?**

A git merge is a local Git operation used to combine changes from one branch into another within a repository.  
A GitHub pull request is a collaboration feature that allows someone to propose changes, review them, discuss them, and then merge them into another branch through the GitHub web interface.
